### Phase 1ブリーフィングとの比較
Phase 1では、関係のないパッチ情報に引っ張られたりロール外ピックへの過剰反応（ノイズ）で分析が散漫になる問題が確認されていました。
Step 0の導入（Relevance Pruning）により、今回は:
- **パッチノイズの完全除去**: 相手5人に無関係な多数のチャンピオンバフ/ナーフの情報をLLMが読まなくなったことで、「相手はバフされたAを出してくるかもしれない」といった幻覚・過剰推測が消滅しました。
- **ロール外ピックのペナルティ処理**: だれでものEliseなど、怪しいピックは「意図的な仕込みの可能性がある」として意図的に保持し、その他の散発的なピックは候補から自動足切りされるため、焦点が非常にクリア（勝率やKDAに裏打ちされた真のプールとパッチの交差点のみ）になっています。
