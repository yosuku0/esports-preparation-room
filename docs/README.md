# docs

This directory contains documentation for the **eSports Preparation Room** project.

## Public Documents

| File/Folder | Description |
|---|---|
| [`tester_package/`](./tester_package/) | Onboarding materials for testers (input templates, feedback forms, etc.) |
| [`design-decisions.md`](./design-decisions.md) | Key architectural decisions |
| [`riot_api_analysis.md`](./riot_api_analysis.md) | Riot API endpoint research and usage notes |
| [`patch_source_analysis.md`](./patch_source_analysis.md) | Patch notes data source analysis |
| [`integration_test_verification.md`](./integration_test_verification.md) | Integration test results |

## Private (not tracked in git)

Briefings, internal verification reports, and analysis files containing real player data are stored locally in `archive/` and are excluded from this repository.
