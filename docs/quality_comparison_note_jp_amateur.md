# Phase 1 品質比較ノート (vs Phase 0 & T1)

## 評価観点

### 観点1: 特徴の検出力
- **自己評価**: T1テスト（全く特徴が見えなかった状態）から劇的に改善が見られた。アマチュア特有の「特定の得意チャンピオン（Zaahen等）を集中的にプレイする」傾向や、「勝てる武器（Ezreal, Karma）と勝てない武器（Nilah, Leona, Nautilus）の明確なコントラスト」を実データから浮き彫りにすることができた。

### 観点2: パッチ接続の質
- **自己評価**: 直接的なバフ/ナーフの対象になったピックは少なかったものの、「Kha'ZixナーフによるADジャングラー（Viego/Graves）の相対的浮上」や、「Mel強化が近接APミッド（Diana/Galio）にとって向かい風になる」というメタの文脈を踏まえた間接的な影響推論を適切に展開できた。T1テストのような「パッチ影響なし＝分析不要」という結論にはならなかった。

### 観点3: BAN/PICK仮説の具体性
- **自己評価**: チャンピオンプールの偏り（SupのKarmaが勝率100%に対し、タンクSupが勝率0%であること等）を根拠として、「特定の武器を奪い、苦手な武器を強制させる」という説得力のあるBAN提案を行うことができた。これはPhase 0の「プール2体のうち1体をBANして崩壊させる」というモックシナリオに匹敵する、実用的かつ具体的な仮説である。

### 観点4: チーム傾向の推定
- **自己評価**: T1テストの際には試合時間が短いことを「ソロキュー特有のサレンダー」と切り捨てざるを得なかったが、アマチュア5名のロールごとの傾向（Topファイター、Mid近接AP、ダイブ構成など）のパズルピースが揃ったことで、「中盤（25〜35分）での集団戦・ダイブ構成」というチームとしての確かな勝ちパターンを推定できた。

### 観点5: ノイズ耐性
- **自己評価**: 8/50試合存在したロール外ピック（TopのAmbessa、MidのGarenなど）や1試合のみのテストピックについて、メインロールの傾向分析に混ぜ込むことなくノイズとして適切に除外・処理した結果、抽出されたインサイトの解像度が非常に高くなった。

## 結論
**Phase 0（モック）≒ 今回のJPアマチュアテスト ＞＞ (超えられない壁) ＞＞ T1テスト**

今回の検証により、本システムは「チャンピオンプールに偏りのあるアマチュア〜セミプロ層の戦績データ」を入力とした場合、Phase 0の精巧に作られたモックデータと遜色ない、**コーチレベルの深く鋭い分析ブリーフィングレポートを実際に生成可能である**ことが完全に証明された。
